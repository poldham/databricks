# You are in RStudio on the Master Node of a Data Bricks Cluster and want to commit and push to Github from the cluster
# See documentation: 
# If unfamiliar with Git read: https://happygitwithr.com/

1. Go To: Tools > Options > Terminal > Uncheck > Connect with Websockets

2. Open a Terminal

In terminal enter:

git config user.email "myemail@something.com"
git config user.name "My Name"

You may want to use the --global flag after config

3. Do something  
  
3a. Commit
Add commit message as usual etc. 

3b. Push
Enter email address used on github
Enter password used on github

3c. Check repo is updated.
